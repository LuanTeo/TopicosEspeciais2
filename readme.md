# Ferramenta de varredura de portas TCP e UDP em Python

## Requisitos para funcionamento
- Python 3.6+
- Scapy (`pip install scapy`)

## Exemplo de uso

- python3 port_scanner.py <target> [-p PORTS] [-t] [-u] [-o OUTPUT] [-f FORMAT]

## Funcionalidades
- Varredura TCP SYN 
- Varredura UDP
- Exportação em formato .txt
